Задание 5 — цифровой отпечаток пальца на основе кода Тардоса.

## Что реализовано

- генерация матрицы `U` для `n = 10` пользователей;
- параметр `c` вводится пользователем, `epsilon = 0.1` фиксирован;
- ввод `c_real` и состава реальной коалиции;
- внедрение ЦОП в LSB одного BMP-контейнера из набора `container1`;
- генерация 10 персональных копий `users/user_XX.bmp`;
- симуляция коалиционной атаки interleaving: каждый бит пиратского слова берется от случайного участника коалиции;
- извлечение пиратского слова из изображения;
- расчет Tardos-score для всех пользователей;
- обнаружение участников как `top-c_real` пользователей по score;
- проверка результата: TP, FP, FN.

## Запуск

Из каталога `lab5`:

```powershell
javac -encoding UTF-8 TardosFingerprintLab.java
java -Dfile.encoding=UTF-8 TardosFingerprintLab
```

Или в Windows:

```powershell
.\run.bat
```

Значения по умолчанию:

- контейнер: `../container1/1.bmp`;
- ключ: `lab5-tardos-key`;
- если список коалиции не вводить, берутся первые `c_real` пользователей.

## Результаты

После запуска создается папка `out`:

- `probabilities.csv` — вероятности `p_j` кода Тардоса;
- `tardos_matrix_U.csv` — матрица `U` размера `10 × m`;
- `users/user_XX.bmp` — изображение с ЦОП конкретного пользователя;
- `pirate.bmp` — пиратская версия после коалиционной атаки;
- `pirate_word.csv` — извлеченное пиратское кодовое слово;
- `scores.csv` — рейтинг пользователей, score, отметка реального участника и обвиненного;
- `summary.txt` — краткое резюме эксперимента.

Длина кода вычисляется как `m = min(W*H, ceil(100*c^2*ln(n/epsilon)))`, чтобы код помещался в одну LSB-плоскость контейнера.
